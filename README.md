# Bond Market Regime Monitor

A live BondStats GitHub tool for classifying the current global sovereign bond market regime using benchmark yield data.

## Overview

The Bond Market Regime Monitor is designed to move beyond simple yield tables and rankings.

It interprets live sovereign yield moves across global bond markets and classifies the current market environment using:

- average daily moves
- cross-market dispersion
- market breadth
- regional direction
- top movers

The tool is built for an institutional-style BondStats interface and is intended to provide a real-time view of whether global bond markets are:

- Stable
- Broad Selloff
- Broad Rally
- Diverging
- High-Volatility Crosscurrents
